## 📌 Summary

GigaNet-1000 is a high‑density network simulation built with Cisco Packet Tracer. It connects 993 end devices (named PC8 … PC1000) through a multi‑layer hierarchical topology. The project showcases:

    Scalable IP addressing (private subnets with VLSM)

    Dynamic routing (OSPF or EIGRP) between distribution/core routers

    VLAN segmentation for broadcast control and security

    Centralized DHCP service for automatic host configuration

    Spanning‑Tree Protocol (STP) for redundant switch links

    Basic ACLs to restrict inter‑VLAN traffic

This simulation serves as a reference for anyone learning how to design, implement, and troubleshoot large networks in a virtual environment.

## ✨ Features

    ✅ 993 end devices (PC8 – PC1000) – each with simulated traffic capability

    ✅ Modular topology – access, distribution, and core layers

    ✅ Dynamic routing – OSPF (or EIGRP) for automatic route exchange

    ✅ VLANs – department/floor‑based segmentation (e.g., VLAN 10, 20, 30)

    ✅ DHCP server – IP address, default gateway, and DNS assignment

    ✅ Spanning‑Tree Protocol – redundant links without loops

    ✅ Access Control Lists – basic security between VLANs

    ✅ Full Packet Tracer file (.pkt) – ready to open and simulate

    ✅ Documentation – IP planning table, VLAN mapping, device configuration examples

## 🛠️ Built With

    Cisco Packet Tracer – version 8.x (or later)

    CLI – for manual configuration of routers, switches, and end devices

    (Optional) Python – if you used scripts to generate configs or IP addresses
